Instalação

Para realizar a instalação antes é necessário adquirir o arquivo comprimido do Curupira. O arquivo poderá ser obtido pelo GitHub ou diretamente pelo link.

https://github.com/apsb/curupira

De posse do arquivo comprimido, extraí-lo em uma pasta qualquer. (Ex. c:\curupira).

Dentro do navegado, clicar em para abrir o gerenciador de extensões.

Clique no botão "Carregar Extensão Expandida ..." é selecione a pasta com os arquivos.

Login

Para proteger o acesso as configurações e identificar o usuário foi desenvolvida uma tela para que o usuário possa digitar uma senha, tornando o acesso as informações controlado. A senha padrão de acesso é "admin". Para utilizar basta digitar a senha e clicar em "Login".

Status

A tela de Status exibe informações sobre a quantidade de Categorias, Sites e Palavras existentes no banco de dados e a data da última atualização.

Preferência

A tela de Preferência possibilita modificar o estado de ativação da extensão, modificar as opções de pesquisa, selecionar as categorias que devem ser bloqueadas, personalizar os sites e palavras a serem bloqueadas, selecionar as categorias pela faixa etária e modificar a senha de acesso.

  1. Configurações de bloqueio, permite escolher onde aplicar as regras de sites e palavras cadastradas.
  2. Horário de acesso, configura a faixa de horário permitida para acesso a internet como um todo.
  3. Seleção de Categorias por Faixa, permite selecionar as categorias usando um cadastro pré estabelecido de Faixa Etária, selecionando automaticamente as categorias correspondentes.
  4. Categorias, ao selecionar a categoria, informa que os sites e palavras dessa vinculados deverão ser bloqueados pela extensão.
  5. Botão Personalizar, exibe um formulário para cadastrar sites e palavras definidas pelo usuário.
  6. Botão Update, recupera do servidor externo as atualizações de sites, categorias, palavras e faixas.
  7. Senha, permite ao usuário modificar a senha atual.
  8. Botão Mudar Senha, verifica a consistência das senhas e modifica a senha atual para a nova senha escolhida.

Relatório

A tela de Relatório permite visualizar os sites que foram acessados, indicando os sites bloqueados.

  1. Filtro, Data Inicial, seleciona os sites acessados a partir dessa data.
  2. Filtro, Data Final, seleciona os sites acessados até essa data estabelecida.
  3. Botão Pesquisar, executa a pesquisa dos sites, conforme os parâmetros estabelecidos.
